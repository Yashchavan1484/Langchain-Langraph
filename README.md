# LangChain & LangGraph Projects

This repository contains my hands-on projects, experiments, and implementations built while learning **LangChain**, **LangGraph**, **RAG (Retrieval-Augmented Generation)**, **AI Agents**, and related LLM application development concepts.

## About This Repository

The goal of this repository is to document my learning journey and showcase practical implementations of modern AI application development using:

* LangChain
* LangGraph
* Retrieval-Augmented Generation (RAG)
* Vector Databases (Pinecone, Chroma, etc.)
* Google Gemini Models
* LangSmith
* Tavily Search
* Streamlit
* Agentic Workflows

## Repository Structure

Each project is organized in its own folder.

